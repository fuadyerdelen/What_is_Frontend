What is Frontend?

What is HTML?

Html (Hypertext Markup Language) is the system used in the preparation of web pages. Browsers such as Chrome, Firefox and Internet Explorer process html codes and convert these codes into web pages.

What is CSS?

CSS (Cascading Style Sheets) allows us to add color, fonts, position etc. properties of web pages. It allows us to play with these features as we wish.

What is JavaScript?

JavaScript makes web pages dynamic. It helps to control the forms on the web pages, to make animations, to add functions to the buttons.

What Is a Front End Developer?

A front end web developer is the person who implements web designs through coding languages like HTML, CSS, and JavaScript.
